# Django 项目目录说明文档

## 介绍
### 快速开始
1.环境准备 python3.8

2.安装依赖
``` bash
pip install -r requirements.txt
```
3.运行
```bash
python main.py
```
或
```bash
# 进入website目录 启动django程序
python3 manage.py runserver 0.0.0.0:8000
```
4.访问
初始用户：admin
初始密码：123456
